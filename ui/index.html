How to play
  • Connect wallet (Extension, Desktop, or WalletConnect QR), then click Start Ranked Run.
  • Starting a run submits the current entry fee on-chain and locks this run to your wallet.
  • You get 5:00 per run. The timer starts only after start-run signing/tx succeeds.
  • Play normally (keyboard arrows, swipe, or tap controls). Your moves and timings are verified on submit.
  • When the board is stuck (or you tap Quit and Submit), use Verify + Submit to finalize.
  • You must use the same wallet for start and submit; switching wallets will fail verification.
  • If your score beats current #1, payout is 50% of pot and the remaining 50% carries forward.
  • When a new #1 is set, next entry fee is reduced by half, with a floor of 1 ALPH.
  • 24h reset is based on the last time someone takes #1: if no new first place is set for 24h, fee/top-score cycle resets on next start.
