How to play
  • Connect wallet, then click Start Ranked Run.
  • You have 5 minutes to finish and submit that run.
  • Use the same wallet for start and submit.
  • 24h Reset is inactivity-based: if no one starts for 24h, entry fee/top-score cycle resets on next start.
